out of order = не работает / неисправен = phrase. not working correctly (often used on signs about machines)
to need fixing = нуждаться в ремонте = phr.v. to need to be repaired
no reception = нет сети / нет сигнала = noun. no mobile signal available (so the phone can't connect to the network)

to keep making (a noise) = постоянно издавать (звук) = phr.v. to repeatedly produce a sound again and again
to crash = вылететь / аварийно завершиться = verb. (about software) to stop working suddenly
to freeze = зависнуть = verb. (about a device/program) to stop responding
not to work any more = больше не работает = phrase. to stop functioning (after working before)
to sort it out = разобраться / решить проблему = phr.v. to fix a problem by dealing with it and finding a solution
to save (a file) onto a memory stick = сохранить (файл) на флешку = phr.v. to copy a file to a USB flash drive
to print something = распечатать (что-то) = phr.v. to produce a paper copy using a printer
